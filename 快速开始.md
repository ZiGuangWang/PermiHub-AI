# 🚀 快速开始 - 5 分钟运行项目

> 适合已有 Node.js 和 MongoDB 的用户

---

## ✅ 前提条件检查

```bash
# 检查 Node.js
node -v  # 应该显示 v18+

# 检查 MongoDB 服务
sc query MongoDB  # 应该显示 RUNNING
```

---

## 📦 第 1 步：安装依赖

### 后端
```bash
cd F:\AI Project\backend
npm install
```

### 前端
```bash
cd F:\AI Project\frontend
npm install
```

---

## ⚙️ 第 2 步：配置环境变量

编辑文件：`F:\AI Project\backend\.env`

```env
PORT=3000
NODE_ENV=development
MONGODB_URI=mongodb://localhost:27017/permission_manager
JWT_SECRET=permission-manager-jwt-secret-key-2024
JWT_EXPIRES_IN=24h
LOG_LEVEL=info
```

> 注意：如果 3000 端口被占用，可以修改为其他端口（如 3001、3002 等）

---

## ▶️ 第 3 步：启动服务

### 启动后端
```bash
cd F:\AI Project\backend
npm run dev
```

看到 `✓ MongoDB 连接成功` 表示成功。

### 启动前端
**新开一个命令行窗口**：
```bash
cd F:\AI Project\frontend
npm run dev
```

看到 `Local: http://localhost:5174/` 表示成功。

> 注意：如果 5173 端口被占用，Vite 会自动使用 5174 或其他端口

---

## 🌐 第 4 步：访问系统

浏览器打开：**http://localhost:5174**

> 如果前端使用的是其他端口（如 5173、5175 等），请以实际启动时显示的端口为准

---

## 👤 第 5 步：创建管理员账号

按 `F12` 打开浏览器控制台，粘贴执行：

```javascript
fetch('http://localhost:3000/api/auth/register', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    username: 'admin',
    password: 'Admin@123',
    email: 'admin@example.com'
  })
}).then(r=>r.json()).then(console.log).catch(console.error)
```

> 注意：如果后端使用的是其他端口，请修改 URL 中的端口号

看到 `{code: 200, ...}` 表示成功。

**登录账号**：
- 用户名：`admin`
- 密码：`Admin@123`

---

## ⚠️ 遇到问题？

### MongoDB 未启动
```cmd
net start MongoDB
```

### 端口被占用
- 后端：修改 `backend\.env` 的 `PORT=3000` 为其他端口（如 3002、3003）
- 前端：Vite 会自动检测并使用可用端口，无需手动配置

### 依赖安装失败
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

完整安装指南请查看：`安装指南.md`
