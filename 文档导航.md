## 🎓 快速上手流程

### 第 1 步：环境准备（10 分钟）
- ✅ 确认 Node.js 已安装：`node -v`
- ✅ 安装 MongoDB（参考 `安装指南.md`）
- ✅ 确认 MongoDB 服务运行：`sc query MongoDB`

### 第 2 步：启动项目（5 分钟）
- ✅ 启动后端：`cd backend && npm run dev`
- ✅ 启动前端：`cd frontend && npm run dev`
- ✅ 访问前端：http://localhost:5174

### 第 3 步：创建管理员（2 分钟）
- ✅ 浏览器打开控制台（F12）
- ✅ 执行注册代码（见 `快速开始.md`）
- ✅ 使用 admin/Admin@123 登录

### 第 4 步：熟悉功能（20 分钟）
- ✅ 创建系统
- ✅ 创建权限
- ✅ 创建角色
- ✅ 创建账号并分配权限

## 🔧 常用命令速查

### MongoDB 相关
```bash
sc query MongoDB         # 检查服务状态
net start MongoDB        # 启动服务
net stop MongoDB         # 停止服务
```

### 后端相关
```bash
cd backend
npm install             # 安装依赖
npm run dev             # 开发模式启动
npm start               # 生产模式启动
```

### 前端相关
```bash
cd frontend
npm install             # 安装依赖
npm run dev             # 开发模式启动
npm run build           # 构建生产版本
```

## 📞 获取帮助

### 查看日志
- **后端日志**：后端命令行窗口输出
- **前端日志**：浏览器 F12 → Console
- **MongoDB 日志**：`F:\MongoDB\Log\mongod.log`

### 检查服务
```bash
# 检查 MongoDB
sc query MongoDB

# 检查 Node.js
node -v

# 检查端口占用
netstat -ano | findstr :3000
netstat -ano | findstr :5174
```

### 重启所有服务
```bash
# 1. 重启 MongoDB
net stop MongoDB && net start MongoDB

# 2. 重启后端（新窗口）
cd backend && npm run dev

# 3. 重启前端（新窗口）
cd frontend && npm run dev
```

## ⚠️ 重要提示

1. **端口配置**
   - 后端：3000（如被占用可修改 `backend\.env`）
   - 前端：5174（Vite 自动检测）
   - MongoDB：27017

2. **MongoDB 说明**
   - 自动创建数据库和集合
   - 无需手动执行建表语句
   - 可使用 `init-database.js` 创建测试数据

3. **管理员账号**
   - 需要通过浏览器控制台创建
   - 或使用 `init-database.js` 初始化
   - 密码通过 bcrypt 加密存储
