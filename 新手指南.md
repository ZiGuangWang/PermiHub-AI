# 🚀 零基础上手指南 - 多系统权限管理系统

## 📖 本指南适合谁

- ✅ 已安装 Node.js
- ❌ 未安装 MongoDB
- ❌ 未安装 Docker
- 💡 想要最简单快速启动项目

---

## 🎯 最快速启动方案（5 分钟）

### 第一步：获取 MongoDB 云数据库（3 分钟）

1. **打开浏览器访问**：https://www.mongodb.com/cloud/atlas/register
   
2. **注册账号**
   - 使用邮箱注册
   - 或使用 Google/GitHub 账号快捷登录

3. **创建免费集群**
   - 选择 "M0 Sandbox"（免费）
   - 选择 "AWS" 提供商
   - 选择离你最近的区域（如：Singapore 新加坡）
   - 点击 "Create Cluster"

4. **获取连接字符串**
   - 等待集群创建完成（约 1-2 分钟）
   - 点击 "Connect" 按钮
   - 选择 "Connect your application"
   - 选择 "Node.js" 驱动
   - 复制连接字符串（格式类似：`mongodb+srv://username:password@cluster0.xxx.mongodb.net/`）

### 第二步：运行启动脚本（2 分钟）

1. **在项目文件夹打开命令行**
   - 打开文件资源管理器
   - 进入 `f:\AI Project` 文件夹
   - 在地址栏输入 `cmd` 并回车

2. **运行快速启动脚本**
   ```bash
   quick-start.bat
   ```

3. **按提示操作**
   - 当询问是否已配置 MongoDB 时，输入 `Y`
   - 粘贴刚才复制的 MongoDB 连接字符串
   - 等待依赖安装完成
   - 自动启动后端和前端服务

4. **访问系统**
   - 浏览器打开：http://localhost:5173

## 🔧 分步启动方案（如果脚本启动失败）

### 步骤 1：配置 MongoDB

编辑 `backend\.env` 文件（用记事本打开）：

```env
PORT=3000
NODE_ENV=development
MONGODB_URI=你的 mongodb+srv 连接字符串
JWT_SECRET=permission-manager-jwt-secret-key-2024
```

### 步骤 2：启动后端

```bash
# 打开命令行
cd f:\AI Project\backend

# 安装依赖
npm install

# 启动服务
npm run dev
```

看到以下输出表示成功：
```
MongoDB 连接成功：xxx
服务器启动在端口 3000

# 安装依赖（如果已安装可跳过）
npm install

# 启动服务
npm run dev
```

看到以下输出表示成功：
```
VITE ready in xxx ms
➜  Local:   http://localhost:5173/
```

## 👤 创建第一个管理员账号

由于登录页面没有注册入口，需要通过以下方式创建账号：

### 方法一：使用浏览器控制台（推荐）

1. **打开浏览器开发者工具**
   - 访问 http://localhost:5173
   - 按 `F12` 键

2. **切换到 Console 标签**

3. **复制粘贴以下代码并回车**：

```javascript
fetch('/api/auth/register', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    username: 'admin',
    password: 'Admin@123',
    email: 'admin@example.com'
  })
})
.then(response => response.json())
.then(data => console.log('注册结果:', data))
.catch(error => console.error('注册失败:', error));
```

4. **查看结果**
   - 看到 `{code: 200, data: {...}}` 表示成功
   - 关闭开发者工具

5. **登录系统**
   - 用户名：`admin`
   - 密码：`Admin@123`

### 方法二：使用 Postman/Apifox

1. **下载 Apifox**（推荐中文界面）
   - 访问：https://apifox.com/
   - 下载并安装

2. **创建请求**
   - 方法：POST
   - URL：http://localhost:3000/api/auth/register
   - Headers：`Content-Type: application/json`
   - Body（JSON）：
   ```json
   {
     "username": "admin",
     "password": "Admin@123",
     "email": "admin@example.com"
   }
   ```

3. **发送请求**
   - 点击 "Send"
   - 看到响应 code 为 200 表示成功

## ⚠️ 常见问题解决

### Q1: 后端启动失败，提示 MongoDB 连接错误

**错误信息**：
```
MongoDB 连接错误：connect ECONNREFUSED
```

**解决方案**：
1. 检查 `.env` 文件中的 `MONGODB_URI` 是否正确
2. 确保使用的是 `mongodb+srv://` 开头的云数据库连接字符串
3. 检查网络连接（Atlas 需要访问外网）

### Q2: 前端启动失败，提示端口占用

**错误信息**：
```
Port 5173 is already in use
```

**解决方案**：
1. 编辑 `frontend\vite.config.js`
2. 修改 `port: 5173` 为 `port: 5174`
3. 重新启动前端

### Q3: 接口请求失败（网络错误）

**解决方案**：
1. 确认后端服务已启动（查看命令行是否有报错）
2. 检查 `vite.config.js` 中的代理配置
3. 清除浏览器缓存（Ctrl+Shift+Delete）

### Q4: npm install 失败

**错误信息**：网络超时或权限错误

**解决方案**：
```bash
# 使用淘宝镜像加速
npm config set registry https://registry.npmmirror.com

# 重新安装
npm install
```
